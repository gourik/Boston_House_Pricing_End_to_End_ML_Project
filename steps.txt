1.Create repository in Github account
2.Copy link from Code and open cmd and change path to G:\revision_ml\end_to_end_ml_projects and then run command:
  git clone https://github.com/gourik/Boston_House_Pricing_End_to_End_ML_Project.git
3.Now this creates file in path G:\revision_ml\end_to_end_ml_projects\Boston_House_Pricing_End_to_End_ML_Project
4.install VS Code..and rest of the steps are written in VS Code
5.Create a new environment for each new project.
  change to command prompt
  conda create -p venv python==3.7 -y
6.create requirement.txt and install all libraries: 
  pip install -r requirements.txt
7.We need to push code from VS code to GIT,hence we need to configure Git Cli with Github repository.
	git config --global user.name "gourik"
        git config --global user.email "ugourik@gmail.com"
8.Adding files to GIT:
  git add requirements.txt
  git add.
9.Commit pushes files from local system to Staging environment
    git commit -m "This commit has requirements.txt and README.md files"
10.Using push to publish local commits:
   git push <remote><branch>
   git push origin main
11.Create app.py to create end to end project
12.Create template folder,as render_template() looks for templates folder, and inside this folder create a file called 
   home.html.
13.Now, execute app.py:
   python app.py
